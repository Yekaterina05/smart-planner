# Smart Planner Web Application

A user-centered web application for task planning and productivity.

## Overview
Smart Planner is a lightweight task management web app focused on clarity, usability, and clean user flows.

## Problem
Many users struggle with organizing daily tasks in a simple and intuitive way.

## Solution
A minimal task planner that allows users to:
- Create and manage tasks
- Track task completion
- Use the app across devices

## UX/UI
- User flows and wireframes designed in Figma
- Focus on usability and clean interface
- Reusable UI components

## Tech Stack
- React
- TypeScript
- Firebase (Authentication, Firestore)

## Features
- User authentication
- Real-time database
- Responsive layout
